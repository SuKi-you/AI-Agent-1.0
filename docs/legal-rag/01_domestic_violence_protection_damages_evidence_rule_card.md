# 家暴、人身安全保护与离婚损害赔偿证据规则卡

## 检索关键词

他打我；他家暴我；他动手；被丈夫打；被妻子打；长期挨打；推搡；掐脖子；拳打脚踢；砸东西；威胁我；恐吓我；扬言伤害我；我害怕；不敢回家；报警；出警；派出所；询问笔录；告诫书；伤情；受伤照片；医院诊断；病历；医疗费；人身安全保护令；禁止靠近；禁止骚扰；离婚损害赔偿；想离婚；感情破裂；想要孩子跟我生活；抚养权；孩子害怕对方；对方有暴力倾向。

## 适用诉求

- 离婚
- 家暴 / 人身安全保护
- 离婚损害赔偿
- 子女抚养权

## 相关法律关系

本专题通常同时涉及离婚关系、家庭暴力与人身安全保护关系、离婚损害赔偿关系以及离婚后子女抚养关系。家庭暴力线索可能用于说明夫妻感情是否确已破裂、是否存在现实人身安全风险、无过错方是否主张损害赔偿，以及子女由哪一方直接抚养更有利于未成年人利益。不同诉求的证明对象并不相同，不能仅因用户陈述存在家暴，就直接认定离婚、保护令、损害赔偿或抚养权请求当然成立。

## 法律要件与待证事实映射

| 法律要件/判断点 | 待证事实 | 说明 |
|---|---|---|
| 离婚诉求的主体基础 | 是否存在合法婚姻关系 | 需证明双方存在婚姻登记关系，作为离婚、离婚损害赔偿等婚姻家事诉求的基础。 |
| 家庭暴力事实判断 | 用户主张的家庭暴力行为是否存在并有证据支持 | 需围绕行为时间、地点、方式、伤害后果、报警或求助经过等事实形成证明链条，不能把单方陈述直接写成已认定事实。 |
| 夫妻感情破裂判断 | 是否能够证明家暴事实对夫妻感情破裂具有影响 | 家暴线索可作为感情破裂的重要因素，但仍需结合婚姻持续状况、分居、沟通修复可能等情况综合判断。 |
| 人身安全保护必要性 | 是否存在现实人身安全风险 | 需证明正在遭受或面临家庭暴力、威胁、骚扰等现实危险，保护令更关注紧迫风险和防护必要性。 |
| 损害赔偿基础 | 是否存在身体损害、精神损害或财产损失 | 离婚损害赔偿通常需要证明过错行为、损害后果及两者之间的关联性，不能只凭“家暴”二字直接推出赔偿成立。 |
| 子女抚养利益判断 | 家暴事实是否可能影响子女抚养权判断 | 暴力行为如影响子女安全、心理健康或家庭环境稳定，可能成为抚养权判断因素，但不是唯一因素。 |
| 子女基本情况 | 是否能够证明子女身份、年龄和长期照顾情况 | 子女年龄、户籍、入学、生活安排、主要照顾人等事实会影响抚养安排和未成年人利益判断。 |
| 直接抚养能力 | 用户是否具备抚养能力和稳定生活环境 | 需证明收入、住所、工作时间、照护安排、教育医疗支持等条件，避免只根据用户意愿判断抚养权。 |

## 待证事实与证据映射

### 核心证据

| 证据名称 | 证据类型 | 证明目的 | 对应待证事实 | 适用场景 | 备注 |
|---|---|---|---|---|---|
| 结婚证或婚姻登记信息 | 书证 | 证明双方存在合法婚姻关系 | 是否存在合法婚姻关系 | 离婚、离婚损害赔偿 | 可使用结婚证、婚姻登记档案或民政部门查询材料。 |
| 报警记录 / 出警记录 / 询问笔录 | 书证 | 证明曾因家庭暴力、威胁或冲突向公安机关求助及处理经过 | 用户主张的家庭暴力行为是否存在并有证据支持；是否存在现实人身安全风险 | 家暴、人身安全保护、离婚、损害赔偿 | 需注意记录内容是否载明时间、地点、双方陈述、伤情或处理意见。 |
| 医院诊断证明 / 病历 / 医疗费票据 | 书证 | 证明受伤情况、就医经过、治疗费用和损害后果 | 是否存在身体损害、精神损害或财产损失；用户主张的家庭暴力行为是否有证据支持 | 家暴、离婚损害赔偿 | 医疗材料能证明损害后果，但仍需与施暴行为建立关联。 |
| 伤情照片或录像 | 视听资料 | 展示伤痕、现场状况或暴力行为过程 | 用户主张的家庭暴力行为是否存在并有证据支持；是否存在身体损害 | 家暴、人身安全保护、损害赔偿 | 应保留拍摄时间、原始文件和来源说明，避免剪辑造成真实性争议。 |
| 子女出生医学证明 / 户口本 | 书证 | 证明子女身份、亲子关系、年龄和户籍情况 | 是否能够证明子女身份、年龄和长期照顾情况 | 子女抚养权 | 子女年龄会影响抚养规则和未成年人利益判断。 |
| 用户收入证明 / 居住证明 / 劳动合同 / 工资流水 | 书证 | 证明用户抚养能力、收入稳定性和居住条件 | 用户是否具备抚养能力和稳定生活环境 | 子女抚养权 | 可结合社保、公积金、租赁合同或房产材料形成稳定生活环境证明。 |
| 保护令申请书、危险情况说明、既往报警和伤情材料 | 书证 | 证明申请人面临现实危险并具有采取保护措施的必要性 | 是否存在现实人身安全风险 | 人身安全保护 | 保护令材料应围绕禁止暴力、骚扰、跟踪、接触等具体保护需求整理。 |

## 易混淆点

- 用户说“他打我”不等于已经存在报警记录或就医记录。
- 用户说“他家暴我”不等于法院已经认定家暴事实。
- 家暴线索可能影响离婚、保护令、损害赔偿和抚养权判断，但不能直接承诺结果。
- 家暴事实与离婚损害赔偿之间还需要证明过错、损害后果及关联性。
- 家暴事实对抚养权有影响，但仍需结合子女年龄、长期照顾情况、双方抚养能力等因素。
- 医院材料通常能证明伤情和治疗费用，但不必然单独证明伤害由对方实施。
- 聊天记录、录音录像等材料应关注真实性、完整性、合法性和关联性。

## 不应直接推出的结论

- 不得仅凭用户陈述就写成“已有报警记录”。
- 不得仅凭用户陈述就写成“已有医院诊断证明”。
- 不得承诺一定判决离婚。
- 不得承诺一定取得子女抚养权。
- 不得承诺一定支持离婚损害赔偿。
- 不得把“用户主张家暴”写成“法院已认定家暴”。
- 不得诱导用户非法录音、偷拍、窃取他人隐私材料。
- 不得替代律师作出正式法律判断。
